Detalhe do jogo

Toma parte em ferozes batalhas de comando 3-on-3! Conduza seu Digimon para a vitória como fazer uso de suas Leader Skills, Signature Skills and Legacy Skills! Com uma consciência de seus atributos e Resistências para oprimir seus inimigos.

Leader Skills

São Habilidades que o Digimon pode ter ou não, caso possua, basta usá-lo como Líder para ativar a LS. Serve para aumentar um status de acordo com seu tipo.

Legacy Skills

Os Digimons possuem duas Habilidades, específica (skill principal) e de herança (skill secundária). É possível alterar a habilidade de herança, transferindo de outro digimon.

Signature Skills

É a Skill primária, que sempre muda conforme o Digimon evolui.

Estágios de Evolução

São as diferentes fases do desenvolvimento e maturação dos Digimons. Existem 6 níveis correntes.

Bebé I (In-Training )
O Nível I é denominado Bebé I. Os Digimons deste estágio não apresentam estruturas desenvolvidas ou organizadas. Possuem diminutas capacidades ofensivas e normalmente dependem da ajuda de outros Digimons ou humanos para sobreviver.

Bebé II (In-Training II)
O Nível II é denominado Bebé II. Os Digimons deste estágio são mais fortes que antes, e é normal que desenvolvam membros, tornando-se mais móveis que a sua anterior forma gelatinosa.

Criança (Rookie)
O Nível III é denominado Criança. Os Digimons deste estágio são os mais populares e conhecidos, tendo forte destaque em todos os mídias. Esta forma é apelativa porque é suficientemente juvenil para ser considerada fofa e conta com ataques e desenhos muito distintos.

Adulto (Champion)
O Nível IV é denominado Adulto. Praticamente todos os Digimons chegam a este estágio, mas ultrapassá-lo é um desafio. Os Digimons Adulto são os mais comuns de encontrar em todos os mídia, para além dos animes.

Perfeito (Ultimate)
O Nível V é denominado Perfeito. Os Digimons deste estágio sobreviveram à parte mais desgastante da evolução e estão agora entre os seres mais poderosos.

Final (Mega)
O Nível VI é denominado Final. Os Digimons deste estágio são aqueles que atingiram o pico da evolução. São raros e tendem a ser extraordinariamente poderosos.

Recuperação de conta(Código)

Para garantir que sua conta não seja perdida, é importante conecta-la com o Google/ICloud e gerar um código de recuperação, caso tenha perdido sua conta antes de gerar um código ou o mesmo venceu, pule para o passo 2.

1. Gerando o código de recuperação.
1.1. Para as opções de recuperação, você deve selecionar a última opção do menu, chamada de "Others".
1.2. Em "Others" vai aparecer outro menu, lá você deve selecionar "Transfer".
1.3. Em "Transfer" vai aparecer mais duas opções, para gerar o código, selecione a primeira.
1.4. Na tela de Recuperação, basta selecionar "Gerar Código" e pronto!
1.5. Para utilizar o código, é só ir na tela inicial do jogo e selecionar "Transfer"
1.6. Outra tela deve abrir, basta escrever os códigos e apertar em "confirmar"!

2. Recuperando uma conta já perdida.
Para recuperar a conta se não possui o código, é necessário mandar um Email para o Linkz, o passo a passo você confere aqui.
