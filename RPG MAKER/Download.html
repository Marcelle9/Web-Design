Simples até para uma criança, poderoso para um desenvolvidor!

RPG MAKER VX ACE melhora todos os aspectos da criação de RPG, sendo não só a mais nova, mas tambem a melhor engine da série RPG Maker. Com suporte a multiplos tilesets, controle de sombras, e um sistema de recursos flexivel, Ace lhe oferece a mais poderosa ferramenta para criação de RPGs.

Ferramentas

Ace oferece todas as ferramentas que você precisa para fazer o jogo dos seus sonhos. Faça o seu jogo, do seu jeito, com os seguintes recursos:

  • Um editor de mapa aprimorado para construir o seu mundo.
  • Um banco de dados com Personagens/Inimigos/Habilidade/ e com um novo e poderos, sistema de caracteristicas.
  • Novas opções de eventos, incluindo tocar vidos Ogg Theora.
  • Uma maior quantidade graficos padrão, incluindo um Criador de Personagens para criar os proprios heróis e vilões.
